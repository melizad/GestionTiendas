# Sistema de Gestión de Tiendas

Este es un programa de consola desarrollado en **C#** diseñado para administrar los procesos básicos de una tienda de barrio. 
Permite la gestión de usuarios, control de inventario de artículos y procesamiento de ventas.

## Credenciales por defecto

Al iniciar el programa por primera vez, puedes acceder con:
* **Usuario (Cédula):** `00000`
* **Contraseña:** `12345`
